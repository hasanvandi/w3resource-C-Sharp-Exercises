یک برنامه سی شارپ بنویسید که خروجی جمع، تفریق، ضرب و تقسیم دو عدد وارد شده توسط کاربر را روی صفحه نمایش چاپ کند.

داده‌های آزمایشی:
عدد اول را وارد کنید: ۲۵
عدد دوم را وارد کنید: ۴
خروجی مورد انتظار:
۲۵ + ۴ = ۲۹
۲۵ - ۴ = ۲۱
۲۵ x ۴ = ۱۰۰
۲۵ / ۴ = ۶
۲۵ mod ۴ = ۱
مستند:دو متغیر عددی در یک خط تعریف کرده ایم از کاربر خواسته اعداد را وارد کند و عملیات محاسباتی را
روی دو اعمال رو چاپ کرده ایم
