۸. جدول ضرب

یک برنامه سی شارپ بنویسید که جدول ضرب یک عدد را به عنوان ورودی چاپ کند.

داده‌های آزمایشی:
عدد را وارد کنید: ۵
خروجی مورد انتظار:
۵ * ۰ = ۰
۵ * ۱ = ۵
۵ * ۲ = ۱۰
۵ * ۳ = ۱۵
...
۵ * ۱۰ = ۵۰
مستند:در این تمرین عددی را از کاربر گرفته و بااستفاده از حلقه for آن را در اعداد بین صفر تا 10 ضرب کرده
و روی صفحه نمایش داده ایم از حلقه for برای جلوگیری از تکرار استفاده کرده ایم.
